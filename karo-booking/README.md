# KARO Booking

SaaS-платформа для онлайн-записи клиентов.

## Запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000`.

## Структура

- `src/app` — маршруты Next.js App Router;
- `src/components` — переиспользуемые UI-компоненты;
- `src/features` — изолированные продуктовые модули;
- `src/lib`, `src/services` — инфраструктура и будущая работа с данными;
- `src/hooks`, `src/types`, `src/utils` — общие хуки, типы и утилиты.
