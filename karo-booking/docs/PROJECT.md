# KARO Booking

KARO Booking — SaaS-платформа онлайн-записи для компаний сферы услуг: салонов, клиник, стоматологий, автосервисов и других бизнесов по записи.

## Текущий этап

Реализуется интерфейс на Next.js, TypeScript и Tailwind CSS без базы данных, авторизации, API и AI-интеграций.

## Принципы

- модульная feature-based архитектура;
- серверный рендеринг по умолчанию;
- независимость UI от источника данных;
- готовность к интеграции Supabase и PostgreSQL.
