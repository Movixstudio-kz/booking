# База данных

На текущем этапе база данных не подключена.

Планируемый стек: Supabase + PostgreSQL.

Предварительные доменные сущности: `organizations`, `users`, `staff`, `clients`, `services`, `appointments`, `working_hours` и `locations`.

Каждая бизнес-сущность будет принадлежать организации, чтобы поддерживать мультиарендную SaaS-модель.
